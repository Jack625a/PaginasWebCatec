Hoja de Estilos - CSS


Estilos


CSS son las siglas en inglés para «hojas de estilo en cascada» (Cascading Style Sheets). Básicamente, es un lenguaje que maneja el diseño y presentación de las páginas web, es decir, cómo lucen cuando un usuario las visita. Funciona junto con el lenguaje HTML que se encarga del contenido básico de las páginas. Se les denomina hojas de estilo «en cascada» porque puedes tener varias hojas y una de ellas con las propiedades heredadas (o «en cascada») de otras. Para muchas personas una simple plantilla de blog es suficiente. Aun así, cuando quieras personalizar la apariencia de un sitio necesitarás implementar CSS que, en conjunto con un buen CMS, te ayudará a potenciar el alcance de tu contenido.

Maquetacion Básica

ancho de cualquier elemento: width

alto de cualquier elemento: heigth

alineaciones: vertical-align

margenes: margin

  • Margen parte superior: margin-top
  • Margen parte inferior: margin-bottom
  • Margen parte izquierda: margin-right
  • Margen parte derecha: margin-left
  • Margen a todo el elemento:margin

Espaciados: padding

Estilos para fuentes y textos

  1. Tipo de letra:font-family
  2. Tamaño de letra: font-size
  3. Peso de nuestra letra: font-weight
  4. Estilo del texto: font-style
  5. Decoraciones al texto: text-decoration
  6. Alineacion del texto: text-align
  7. Color del texto:color

Colores y fondos - Estilos

  • Color de textos: color
  • Color de fondo de un elemento: background-color
  • Color de Fondo: background
  • Imagen de Fondo: background-image
  • Sombreado en cajas: box-shadow
