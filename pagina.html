Comandos basicos de GIT

GIT INIT

Inicializa el repositorio

GIT ADD

Lleva los archivos a la zona temporal

GIT COMMIT

Confirma los cambios para una nueva version

GIT STATUS

Muestra el estado en el que se encuentra el repositorio

GIT LOG

Muestra el historial del repositorio

GIT DIFF

Compara diferentes versiones entre 2 repositorios

GIT PUSH

Envia modificaciones de archivos locales a remoto

GIT PULL

Recibe modifcaciones de repositorio remoto a local

GIT BRANCH

Envia al pasado sin alterar el futuro
